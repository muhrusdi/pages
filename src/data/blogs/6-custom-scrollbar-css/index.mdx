scrollbar element yang berada dalam container atau yang biasa disebut nested scroll, atau dikondisi lain ketika
ingin membuat interface kita terlihat lebih konsisten ketika berada dibeberapa sistem operasi.

Berikut beberapa contoh untuk mengubah style scrollbar di browser menggunakan properti css prefix `::webkit-scrollbar`:

> Custom scrollbar menggunakan CSS hanya mendukung di browser yang berbasis `webkit` seperti Chrome/Safari.
